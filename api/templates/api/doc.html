Archive API

Welcome to the Archive API. We've developed a versatile RESTful API for public use. We hope that you can utilize this API in your own projects. Here, you can find a guide to using the API.

The API can be queried through the addresshttp://keelan.pythonanywhere.com/api/. All responses are given in JSON.

Bills

You can access all bills stored in the Archive. You can access them individually, or in specific groups.

http://keelan.pythonanywhere.com/api/billwill return all bills. They are sorted by ID number.

http://keelan.pythonanywhere.com/api/bill/<name>will return the bill with the exact name specified. There is no search functionality yet. The name is exactly as it appears on the Archive website, excluding the bill identifier (e.g. HR.001, S.1004), with underscores replacing spaces. For example, the "Back to the Moon Bill" could be accessed using/api/bill/Back_to_the_Moon_Bill. This parameter is case sensitive.

http://keelan.pythonanywhere.com/api/bill/<house>/<number>will return the bill with the exact house and number specified. To specify the house, you can use the house id, the house code (case insensitive), or the name of the house. Similar to bill names, the name of the house is case sensitive, and must be exactly as it appears on the website, with underscores instead of spaces (e.g. House_of_Representatives). An easy method for this parameter is to take the bill identifier (e.g. HR.001, S.1004) and replace the period with a slash - HR.001 becomes/api/bill/HR/001. You do not need to include leading zeros in bill numbers.

http://keelan.pythonanywhere.com/api/house/<house>/billswill return all bills from the house specified. They are sorted by ID number. To specify the house, you can use the house id (see the section on Houses), the house code (case insensitive), or the name of the house. Similar to bill names, the name of the house is case sensitive, and must be exactly as it appears on the website, with underscores instead of spaces (e.g. House_of_Representatives).
